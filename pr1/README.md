## В рамках першої лабораторної роботи було виконано:
- Встановлено середовище Linux як віртуальну машину
  
  Варіантом лінукса було обрано Linux Mint
- Виконано просту програму для перевірки роботи системи та встановлення компілятора у вигляді hello.c
- Було виконано ще одну просту програму мовою С для пошуку завтрашньої дати

## В якості програм для використання запропоновано два консольні застосунки:
### hello.c
Використання:./hello
В результаті виконання в термінал повинно вивестись повідомлення "Hello Mint".
---
### pr1_v10.c
Використання:./pr1_v10
Програма що видає завтрашню дату відносно введеної.
Для використання пропонується два види календарів - грерогианський та юліцианський. Для обирання календаря необдіхно при запуску ввести 0 або 1 відповідно до бажаного календаря.
Далі необхідно ввести дату у форматі ДД ММ РРРР (день-місяць-рік). (Наприклад: 22 05 2003).
Результатом виконання буде дана наступного дня.
